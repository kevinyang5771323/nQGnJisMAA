# 项目简介
## 核心功能
- 软件开发源码带文档
- 协同过滤推荐算法
## 适用场景
- 农产品销售领域
## 技术特点
- 架构模式：B/S 模式
- 运行环境：Windows
- 开发工具：IDEA + JDK17 + Maven + MySQL5.7/8
- 核心技术：SpringBoot3、Mybatis-Plus、Vue2、MySQL
- 角色设计：
  - 普通用户：商品浏览、下单支付、订单查询、收藏与评价、个性化推荐
  - 农户：商品发布、销售数据统计、订单处理
  - 系统管理员：用户与权限管理、商品审核、订单监管、数据分析
- 前端功能模块：
  - 首页门户：轮播广告、推荐专区、分类导航
  - 商品中心：商品列表、详情展示、关键词搜索
  - 购物车：添加商品、数量调整、移除商品
  - 订单流程：下单 → 支付 → 状态跟踪 → 物流查询
  - 用户中心：个人资料、收货地址、历史订单、我的收藏
  - 评价系统：用户评价、评价管理
  - 智能推荐：协同过滤个性化推荐 + 热销榜单
  - 邮箱注册：支持 QQ 邮箱注册，真实邮箱认证
- 后台管理模块：
  - 用户管理：账号审核与权限分配
  - 商品管理：上架、下架与审核
  - 订单管理：订单处理、发货监管
  - 评价管理：评价内容审核、数据统计
  - 数据看板：销售分析、用户行为洞察
  - 系统设置：轮播图、分类管理
- 项目亮点：
  - 新技术栈：SpringBoot3 + Vue3，前后端分离，性能更强
  - 个性化推荐：协同过滤推荐算法，为用户打造专属推荐，提升复购率
  - 支付可扩展：支付宝沙箱支付，贴近真实业务场景，方便二次开发
  - 数据可视化：数据可视化看板，实时掌握销售与运营情况
  - 源码开放：支持二次开发与个性化定制
